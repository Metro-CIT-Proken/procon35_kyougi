# 操作方法


##自動操作の場合

```
python3 main3.py 問題json 回答json "a" 
```

* UP　キー 左下にズームイン
* DOWN　キー　ズームアウト


* Wキー  上にカメラを動かす
* Sキー　下にカメラを動かす
* Aキー　左にカメラを動かす
* Dキー　右にカメラを動かす
* Rキー　元に戻す


* aオプション 自動で盤面を操作する
* mオプション 手動で盤面を操作する
* a or m オプションに　s を追加することで　ゴールとの距離に応じて色が変化する盤面を表示するようになる

##手動操作の場合

```
python3 main3.py 問題json 回答json  "m"
```

*ズームとカメラの操作は手動と変わらない
